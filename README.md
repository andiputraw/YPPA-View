# Svelte + TS + Vite

front-end untuk YPPA-electron.

cara jalankan

1. Download dependecies yang diperlukan

   ```bash
   npm install
   ```

2. Jalankan di web

   ```bash
   npm run dev
   ```

3. Copy file untuk di gunakan di electron (pengguna mungkin harus mengubah konten dari file `copy_to_electron.bat` )

   ```bash
   ./copy_to_electron
   ```
